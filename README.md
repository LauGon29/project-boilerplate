# Project Boilerplate

This is a basic boilerplate with:
* HTML
* SASS
* ES6
* BABEL
* GULP

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. 
Clone this repo:
```
https://github.com/LauGon29/project-boilerplate.git
```

### Installing

Run the following commands (be sure you are in the correct root)

* Installing dependencies
```
npm install
```

* Running the project: this command is running gulp, and also the folder dist is created at the same time
```
npm start
```

* To update dependecies to latest version, run:
```
npm install -g npm-check-updates
```

```
npm-check-updates
```

```
npm-install
```
